# 火种档案

一个「流浪地球」设计资源站

Vue3 + Nuxt3 + TailwindCSS4 + Vercel

## 😎 本地运行

1. 克隆仓库

```shell
git clone https://github.com/zaona/tinder-archives-center.git
```

2. 安装依赖

```shell
npm install
```

3. 启动本地开发服务器

```shell
npm run dev
```
